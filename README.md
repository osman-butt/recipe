# Dat.js projekt - Yndlingsopskrift i HTML og CSS
I denne afleveringsopgave skal du lave din yndlingsopskrift og publicere den via GitHub Pages. Opskriften skal implementeres med HTML og CSS på baggrund af præsenterede og indlærte teknikker fra undervisningen.
## Krav
Det er et krav, at du anvender følgende:

- HTML og CSS
- HTML5 Semantiske elementer (tags)
- CSS Grid
- Git, GitHub og GitHub Desktop til at dele din kode
- GitHub Pages til at publicere din løsning
- Værktøjer som VS Code og extensions til at udvikle din løsning
 

I din løsning skal der som minimum være:

- En kort beskrivelse af opskriften
- En eller flere lister med ingridienser knyttet til opskriften
- En fremgangsmåde, herunder tilberedningstid
- Tekst, billeder og/ eller andre visuelle elementer som understøtter opskriften

Du kan tilføje elementer som:
- En tabel over næringsindhold for opskriften
- Rating af opskriften
- Tips og tricks
- Links til lignende opskrifter
- Link til køb af indgredienser 
- Andre visuelle elementer
